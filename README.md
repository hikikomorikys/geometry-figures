# Geometric Figures Calculator

Java implementation of geometric figures with area and perimeter calculations.

## Implemented Figures

1. **Rectangle** - requires width and height
2. **Circle** - requires radius
3. **Triangle** - requires three sides that satisfy triangle inequality

## Features

- Area calculation for all figures
- Perimeter calculation for all figures
- Parameter validation
- Exception handling for invalid parameters
- Polymorphism support through abstract class
